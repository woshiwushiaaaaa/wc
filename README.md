# wc
Correctly count the number of characters, number of words, and number of sentences in the imported pure English TXT text.
![avatar](https://wx1.sinaimg.cn/mw1024/0065Oia3gy1fqj51c01q8j30ls0f340d.jpg)
The first line output is the number of characters
The second line output is the number of words
The number of sentences in the third line is the number of sentences
